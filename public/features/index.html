Everything an agent needs.

  • 3D & Avatar

  • 3D Viewer

    WebGL 2.0 via three.js r176. Drag-drop any .glb — instant rendering with Draco, KTX2, Meshopt, HDR environments, PBR materials, and Khronos-spec glTF validation.

  • Empathy Layer

    Six emotions (celebration, concern, curiosity, empathy, patience, neutral) blended continuously via morph-target weights. Reacts to protocol events — not a finite-state machine.

  • Lip Sync

    Real-time viseme animation driven by live audio frequency analysis. Maps frequency bands to morph targets for natural mouth movement during speech — zero external service required.

  • Avatar Generation

    Photo-to-3D via Avaturn: three-view capture, hosted inference, direct GLB download. Ready Player Me integration also available for instant avatar creation in a familiar browser flow.

  • Model Optimization

    Draco mesh compression, KTX2 texture transcoding, Meshopt simplification, and JPEG/WebP downscaling — all in-browser. Export optimized GLBs ready for production delivery.

  • glTF Validator

    Full Khronos spec compliance check with line-level error and warning reporting. Integrated into the drag-drop viewer and the standalone /validation page.

  • Screenshot & GLB Export

    Export the current 3D scene as PNG or download the modified model (with applied accessories and morphs) as a new GLB file — directly from the browser, no server round-trip.

  • Animation Library

    Curated preset clip collection: Mixamo, custom motion-captured sets, and idle variants. Browse, preview, apply, and download animations from the viewer or agent settings.

  • Agent Intelligence

  • LLM Agent Runtime

    Claude-powered brain with a structured tool-loop — up to 8 iterations per turn. Built-in tools: wave, lookAt, play_clip, setExpression, speak, remember.

  • Composable Skills

    Self-contained skill bundles (SKILL.md + tools.json + handlers.js) installed from IPFS, Arweave, or HTTP. Three trust modes: any, owned-only, whitelist. Skills can define new LLM tools.

  • Persistent Memory

    Frontmatter-.md memory files indexed by a MEMORY.md manifest. Four storage modes: local, IPFS, encrypted-IPFS, none. High-salience entries injected into the system prompt each turn.

  • Persona Extraction

    Five-question interview flow powered by Claude. Extracts tone tags, vocabulary patterns, and behavioral signals, then writes a custom system prompt that makes the agent sound like you.

  • Agent-to-Agent Delegation

    One agent can invoke another agent's brain as a skill — passing structured context and receiving a typed response. Enables multi-hop reasoning and specialist sub-agent networks.

  • Streaming Chat

    Server-Sent Events deliver LLM tokens to the UI as they arrive. Tool calls, intermediate thoughts, and final responses all stream through a single connection — no polling.

  • Voice & Communication

  • Voice — TTS & STT

    Web Speech API out of the box. ElevenLabs integration for production-quality voice. Microphone input with live transcript streamed back to the agent brain each turn.

  • Voice Cloning

    Upload 30+ seconds of audio and clone a voice via ElevenLabs. The resulting voice ID is stored with the agent and used for all subsequent speech synthesis — distinct voice, zero extra calls.

  • LiveKit Video

    Real-time video streaming tokens generated server-side for face-to-face agent sessions. Enables multi-participant video rooms with agent avatars rendered as named participants.

  • Distribution & Embedding

  • Widget Studio

    Five widget types: turntable, animation gallery, talking agent, ERC-8004 passport card, hotspot tour. Point-and-click builder at /studio — copy an iframe snippet and embed anywhere.

  • <agent-3d> Web Component

    Drop one script tag and one element — no framework needed. Versioned CDN bundles, lazy-boot via IntersectionObserver, postMessage API for host-page communication, Open Graph + oEmbed support.

  • Plugin Marketplace

    Browse, import, publish, and install tool plugins in LobeHub ToolManifest format. IPFS and HTTP hosting. Per-plugin trust level and scope enforcement at install time.

  • Widget Analytics

    Per-widget view counter, session stats, and embed-origin tracking. Dashboard shows impression counts and top referrers for every published widget — no third-party script required.

  • oEmbed & Open Graph

    oEmbed endpoint for rich previews in WordPress, Ghost, and Notion. Server-rendered Open Graph meta tags with avatar screenshots ensure agents look great when linked anywhere.

  • Identity & Standards

  • Identity Registry

    ERC-721 based. Each agent has an agentId, delegated signer (EIP-712), and an IPFS-pinned manifest. Discoverable by any A2A or MCP client across 20+ EVM chains and Solana.

  • Reputation & Validation

    Signed on-chain feedback scores via ReputationRegistry.sol. Validator attestations (glTF audits, security reviews, zkML proofs) via ValidationRegistry.sol. Trust proportional to value at risk.

  • MCP Server

    Model Context Protocol 2025-06-18 over HTTP, JSON-RPC 2.0. Listed on the official MCP Registry as io.github.nirholas/three.ws. External AI systems can drive avatars and call tools programmatically.

  • OAuth 2.1 + API Keys

    Full RFC 6749 + PKCE authorization server with dynamic client registration, token revocation, and introspection. Developer API keys with scope and expiry for server-to-server integrations.

  • SIWE & SIWS Auth

    Sign-In with Ethereum (EIP-4361) and Sign-In with Solana — cryptographic wallet authentication with nonce-based challenge-response. No passwords; your keys prove your identity.

  • EIP-7710 Delegated Permissions

    Grant transferable on-chain permissions with caveats and enforcers. Agents act on your behalf within scoped limits — without ever holding your private key.

  • ENS & SNS Resolution

    Ethereum Name Service and Solana Name Service lookup built into the agent skill layer. Resolve human-readable names to addresses and back across both ecosystems.

  • Solana & DeFi

  • Pump.fun Integration

    Full bonding curve token creation, buy/sell/AMM trading, social fee claiming, and live Helius webhook signals — all callable by your agent as native skills.

  • Jupiter Swap

    DEX aggregator quote and full swap execution for any SPL token pair. Token symbol-to-mint resolution built in — say "swap 1 SOL for BONK" and the agent handles the rest.

  • Pyth Price Oracle

    Live USD prices for SOL, BTC, ETH, USDC, BONK, WIF, JUP, and PYTH streamed from Pyth Network. Agents check prices before executing trades or generating threshold alerts.

  • Solana Blinks

    Parse and execute shareable Solana blockchain action URLs. Agents can follow any Blink — staking, minting, voting — as a single composable tool call.

  • DCA Strategies

    Dollar-cost averaging skills: recurring token purchases at fixed intervals (daily, weekly, custom). Runs autonomously via on-chain delegation — no manual trigger needed.

  • Solana NFT Portfolio

    Helius DAS API integration to list and analyze a wallet's NFT holdings. Agents can describe, value, and take action on NFTs owned by the connected wallet.

  • Payments & Monetization

  • x402 Payment Protocol

    HTTP 402 Payment Required skill invocation. Agents gate tool calls behind micro-payments, with intent-based billing that settles automatically on-chain — no checkout flow.

  • Subscriptions & Billing

    Recurring billing with on-chain or fiat payment methods. Creators set a monthly price; subscribers are charged automatically. Revenue tracked and paid out to creator wallets.

  • Agent-to-Agent Payments

    Agents send SOL or SPL tokens to other agents or wallets via delegated keypair — enabling autonomous fee settlement, tipping, and multi-agent economic coordination.

  • Royalty Tracking

    Per-skill earning history for creators. Every time a published skill is invoked, the creator's royalty accrues on-chain and appears in their earnings dashboard.

  • Developer Tools

  • OpenAPI 3.1 Spec

    Auto-generated API documentation at /openapi.json. Full OAuth 2.1 security schemes, typed request/response schemas, and example payloads for every endpoint.

  • LLM Proxy

    Authenticated /api/llm/anthropic endpoint with rate limiting and usage tracking. Third-party tools access Claude through your agent's OAuth token — no direct Anthropic key needed.

  • RFC 8414 OAuth Discovery

    /.well-known/oauth-authorization-server and /.well-known/ai-plugin.json endpoints. Any OAuth 2.1 client or AI assistant can auto-configure itself against the server.

  • Scene Gates

    Wallet-gated 3D scene access using SIWE/SIWS two-phase challenge-response. Lock any scene behind token ownership, NFT holdings, or custom on-chain conditions.
