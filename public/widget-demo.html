How to use it

Copy and paste this code into your website. Customize it with data-* attributes.

Available options:

  • data-agent-id: (Required) The ID of the agent to display.
  • data-background: Set the background. Can be transparent (default), dark, or light.
  • data-nameplate: Set to false to hide the agent's name.

Dynamic Loading

The widget script automatically detects new widgets added to the page. Click the button to add another agent.
