Isolated demo workspace. Nothing here touches the production Avaturn pipeline. Goal: prove out a fully open-source replacement before promoting it.

Open-source avatar pipeline

Three parallel paths replace what we currently get from Avaturn. Each demo is end-to-end functional — capture, generate, preview, export GLB. No mocks, no fake data.

Studio · customization editor

Embeds the open-source CharacterStudio (M3-org / Apache-2.0) editor. Pick a base avatar, customize traits, export GLB. Replaces the Avaturn iframe editor.

stack: @m3-org/characterstudio · @pixiv/three-vrm · three.js

Selfie · photo → 3D face mesh

MediaPipe Face Landmarker (Apache-2.0) runs entirely in-browser to build a textured 3D mesh from one selfie. Replaces the Avaturn photo-to-avatar API.

stack: @mediapipe/tasks-vision · three.js · WebGL

Combined · selfie face + Studio body

Stitches the selfie face mesh onto a Studio body GLB and exports a single merged GLB. Closest analogue to the Avaturn end product.

stack: all of the above + GLTFExporter

What's wired

  • MediaPipe Tasks Vision pulled via npm (@mediapipe/tasks-vision). WASM + model files fetch from Google's CDN at runtime.
  • Triangulation indices from tfjs-models (Apache-2.0) — vendored at src/demo/triangulation.js.
  • CharacterStudio is the existing character-studio/ workspace (M3-org, AGPL — under review). Demo expects the dev server at http://localhost:5173; run npm --prefix character-studio run dev in a second terminal.
  • GLB export uses three.js's GLTFExporter from three/addons.
