How it works

From avatar to on-chain agent in four steps.

No 3D experience required. The platform handles the hard parts.

Upload or generate a 3D model

Drag in a GLB file from a 3D artist, import from Ready Player Me, or use Avaturn to generate one from a photo. The viewer validates and renders it in your browser instantly.

Give it a brain and a voice

Write a persona (or let the wizard do it), pick an LLM model, and optionally connect ElevenLabs for a custom voice. Your agent now speaks, gestures, and remembers who it has talked to.

Register on-chain

Mint the agent as an ERC-8004 token. This gives it a permanent ID, a delegated signer wallet, and an immutable record of every action it takes — provable and auditable forever.

Embed it anywhere

Drop <agent-3d src="..."> into any webpage. No framework. No build step. It just works.

On-chain identity

Your agent isn't a PFP.
It's an economic actor.

A PFP is a static image. A three.ws agent is an ERC-8004 token with a live runtime behind it — it can hold funds, earn from skill calls, build a reputation score, and sign transactions on your behalf. The on-chain record is permanent. No company can turn it off.

Stable on-chain identity — an agent ID that never changes, even if you update the avatar
Delegated signer wallet — the agent can transact autonomously within permissions you define
Signed action log — every speak, skill, and memory event is cryptographically recorded
Reputation score — built from real actions, not self-reported. Stakes and royalties follow it
Cross-chain — Base, Ethereum, and Solana. The manifest is pinned to IPFS
