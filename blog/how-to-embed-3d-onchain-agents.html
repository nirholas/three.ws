How to Embed 3D On-Chain Agents on Your Site

Introduction

Embedding 3D on-chain agents on your website can create a more engaging and interactive experience for your users. This guide will walk you through the process step by step.

Step 1: Get Your Agent's Embed Code

First, you'll need to get the embed code for your 3D on-chain agent. You can usually find this on the agent's profile page.

Step 2: Add the Embed Code to Your HTML

Next, you'll need to add the embed code to your website's HTML. This is as simple as copying and pasting the code into the desired location in your HTML file.

Step 3: Customize the Appearance

Finally, you can customize the appearance of your embedded agent using CSS. You can change the size, position, and more to match your website's design.
